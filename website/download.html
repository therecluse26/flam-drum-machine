Verify your download

Every release ships a SHA256SUMS file. Confirming the checksum guarantees the file you downloaded is bit-for-bit the one we published.

macOS / Linux 
# compare against the published SHA256SUMS
shasum -a 256 FlamKit-1.0.0-macOS.zip
# → the hash must match the value in SHA256SUMS
Windows (PowerShell) 
Get-FileHash .\FlamKit-1.0.0-Windows.zip -Algorithm SHA256
# compare the Hash column to the published SHA256SUMS
