How to get involved

Every rung strengthens the project. Start anywhere.

  • 1 · File an issue — found a bug or have an idea? Open an issue. Look for the good first issue label if you want to help.
  • 2 · Join the conversation — ask and answer in Discussions (and Discord at launch).
  • 3 · Submit a kit — package a CC0 kit as flamkit.yaml and share it. See CONTRIBUTING.
  • 4 · Open a pull request — fix a bug, add a feature, improve the docs. Code contributions keep FlamKit free and improving.
